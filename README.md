# Parser

<p>Informe através do console para validar a sequência de caracteres desejada.</p>
<p>A saída será no console através de uma mensagem "Valido" ou "Invalido".</p>
<p>Resultados esperados:</p>

<p>[](){}</p>
<p>Válido</p>
<p>[)]{}</p>
<p>Inválido</p>
<p>[(]){}<></p>
<p>Inválido</p>
<p>(((((([]))))))</p>
<p>Válido</p>
<p>{{[[()]()]}}</p>
<p>Válido</p>
<p>{[[([)]]]}</p>
<p>Inválido</p>
<p><{(())}></p>
<p>Válido</p>
<p>{<[]()[[]]<>>}</p>
<p>Válido</p>
