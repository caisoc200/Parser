# Parser

Informe através do console para validar a sequência de caracteres desejada.
A saída será no console através de uma mensagem "Valido" ou "Invalido".
Resultados esperados:

[](){}
Válido
[)]{}
Inválido
[(]){}<>
Inválido
(((((([]))))))
Válido
{{[[()]()]}}
Válido
{[[([)]]]}
Inválido
<{(())}>
Válido
{<[]()[[]]<>>}
Válido
